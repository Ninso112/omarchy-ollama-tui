# Documentation

## Screenshots

Screenshot placeholder - to add actual screenshots, take a screenshot of the running TUI and save it as `screenshot.png` in this directory.

You can capture a terminal screenshot using tools like:
- `scrot` or `flameshot` on Linux
- Built-in screenshot tools on macOS/Windows
- Terminal recording tools like `asciinema` for animated demos
